# Klinik-Yakin-Bahau-v2
# 🏥 Klinik Yakin Business Suite (ERP)

Sistem pengurusan perniagaan bersepadu yang direka khas untuk **Klinik Yakin Bahau SDN BHD**. Sistem ini menggabungkan fungsi jualan (POS), pengurusan stok, perakaunan double-entry, dan sistem giliran pesakit dalam satu platform web yang moden.

## 🚀 Ciri-Ciri Utama
* **Kasir (POS):** Antaramuka mesra pengguna untuk jualan ubat dan konsultasi.
* **Dashboard Analitik:** Paparan graf prestasi mingguan dan keuntungan masa nyata.
* **Sistem Giliran (Live Queue):** Paparan digital untuk ruang menunggu pesakit.
* **Modul Perakaunan:** Rakaman jurnal automatik (Debit/Kredit) bagi setiap transaksi.
* **Responsif:** Dioptimumkan untuk penggunaan di telefon bimbit dan tablet.
